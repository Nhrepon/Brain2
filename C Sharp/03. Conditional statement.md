## Switch case
```c#
int digit;
switch(digit){

case 0: // case int num when num>1 && num < 5:
Console.Write("Zero");
break;

case 1:
Console.Write("One");
break;

default:
Console.Write("Invalid");
break;
}


// shorthand switch
String message = digit switch{
1 => "One",
2 => "Two",
_ => "not a digit"
}

```

For loop
- initializer; condition; update
```c#
for(int i = 0; i < 50; i++){
	Console.WriteLine($"{i} For loop working!");
}

```

While loop
```c#
int i = 0;
while(i>10){
	Console.WriteLine($"{i} While loop working!");
	i++;
}
```


Do While loop
```c#
int i = 0;
do{
	Console.WriteLine($"{i} While loop working!");
	i++;
}while((i>10));
```
